¡Bienvenido a Programación web 1!

En esta materia se verán los siguientes temas:

  1. Principales conceptos y terminologías relacionadas al desarrollo web.
  2. Programación en HTML.
  3. Diagrama y diseño de una página web.
  4. Desarrollo y aplicación de estilos usando CSS.
  5. Diseño de formularios HTML.
  6. Programación de scripts en el cliente con javascript.
  7. Desarrollo de expresiones regulares.
  8. Validación de formularios con Javascript
  9. Utilización de la Api DOM del navegador.

Estos temas se pondrán a prueba meadiante la realización de trabajos prácticos y, de manera más completa, en la realización de un proyecto final que implica la creación se un sitio web.
