Unidades temáticas

UNIDAD I - LÓGICA

Proposiciones. Tablas de verdad. Funciones proposicionales. Operaciones con proposiciones. Propiedades de las operaciones. Ejecución de acciones condicionadas. Leyes lógicas. Condicionales asociados. Negación de un condicional. Cuantificadores. Circuitos lógicos.

UNIDAD II - CONJUNTO

Introducción. Especificación de conjuntos. Conjuntos especiales. Cardinalidad. Igualdad e Inclusión entre conjuntos. Operaciones entre conjuntos. Propiedades de las operaciones. Producto cartesiano. Relaciones binarias.

UNIDAD III - RELACIONES

Introducción. Relación binaria. Dominio y Rango de una relación. Relación inversa de una relación. Composición de relaciones. Relaciones sobre un conjunto. Propiedades de las relaciones. Relaciones de orden parcial y total. Relaciones de equivalencia. Clases de equivalencia. Algunas relaciones particulares. Representación gráfica de las relaciones definidas sobre conjuntos finitos.

UNIDAD IV - SISTEMA DE ECUACIONES LINEALES Y MATRICES

Sistemas de ecuaciones lineales. Matrices. Solución de un sistema de ecuaciones lineales utilizando matrices. Determinantes. Propiedades. Operaciones matriciales: Suma, Resta, Producto. Matriz Identidad. Inverso de una matriz
